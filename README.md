# googleAuthenticator

## 只是demo

## 使用方法
> + 配置好数据库
> + 启动程序
> + 访问 <a href="localhost:9090/login" title="Title">localhost:9090/login</a>
> + 绑定googleAuthenticator按钮点击后 在日志中查看密钥及二维码生成串，绑定认证器
> + 使用认证器认证，日志中查看结果
